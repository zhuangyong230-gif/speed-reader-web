# 快读领读器 Web

这是可以直接发布到 GitHub Pages 的静态站点。

## 发布到 GitHub Pages

1. 新建一个 GitHub 仓库，例如 `speed-reader-web`。
2. 上传本文件夹里的全部文件。
3. 进入仓库 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`，保存。
6. 等几十秒，GitHub 会生成一个 `https://你的用户名.github.io/speed-reader-web/` 地址。

## iPhone 使用

用 Safari 打开 GitHub Pages 地址，点分享按钮，再点“添加到主屏幕”。
